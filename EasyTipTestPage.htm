Easytipsimple & customizable tooltip plugin

Demo

Position Left:

$("#left").easytip({ easyMessage: "Help to the left", easyPosition: "left" });

Position Above:

 $("#above").easytip({ easyMessage: "Help above the element", easyPosition: "above" });

Position Right:

$("#right").easytip({ easyMessage: "Help to the right", easyPosition: "right" });

Position Below:

$("#below").easytip({ easyMessage: "Help below the element", easyPosition: "below" });

Position Overlay:

 $("#overlay").easytip({ easyMessage: "Help over the element", easyPosition: "overlay" });

Get message text with a function:

 $("#function").easytip({ easyMessage: function(){}, easyPosition: "right" });

Supply element Css:

 $("#css").easytip({ easyMessage: "My tool tip has custom Css to change the text color and weight", easyPosition: "right", easyCss: {"font-weight": "bold", "color": "black"} });
